# 其他小技巧
## 修改终端字体大小

适用于 Ubuntu Server 纯命令行环境（非 SSH 远程连接）。
### 1. 交互式配置 (推荐)

通过系统内置工具重新生成控制台配置。
```bash
sudo dpkg-reconfigure console-setup
```

- **编码 (Encoding)**: 保持 `UTF-8`。
    
- **字符集 (Character set)**: 保持 `Combined - Latin; Slavic cyrillic; Greek`。
    
- **字体 (Font)**: 建议选择 **Terminus** (该字体在高分辨率下表现最清晰)。
    
- **字号 (Size)**:
    
    - 常规屏幕推荐：`8x16` 或 `10x20`。
        
    - 4K 屏幕/视力不佳：推荐 `16x32`。
        

### 2. 命令行手动测试

如果你想在不重启配置的情况下尝试不同字体：
```bash
# 路径通常位于 /usr/share/consolefonts/
sudo setfont /usr/share/consolefonts/Uni2-Terminus32x16.psf.gz
```

## 选择镜像站

使用自动测速脚本，寻找当前网络环境下延迟最低、带宽最大的镜像站。

### 方案一：官方 Python 工具 `apt-select` (推荐)

这是目前最成熟的第三方工具，可以测试延迟并生成新的 `sources.list`。

**安装步骤：**

```bash
sudo apt update && sudo apt install python3-pip -y
pip install apt-select
```

**使用方法：**
```bash
# 列出中国区延迟最低的 5 个镜像站
apt-select --country CN --top 5
```

该工具会输出各站点的延迟时间，并询问你是否生成新的配置文件。

---

### 方案二：利用 `apt-smart` (更智能的切换)

`apt-smart` 能够自动检测当前网络环境下最快的镜像站，并支持一键替换。

**快速运行：**
```bash
# 安装
pip install apt-smart

# 发现当前最快镜像
apt-smart --find-best-mirror

# 直接应用最快镜像
apt-smart --auto-change-mirror
```

---

### 方案三：原生命令行一行指令 (简易测速)

如果你不想安装额外的 Python 包，可以使用这个简单的 Shell 脚本片段来检测常用镜像站的连接速度（以中国区为例）：
```bash
MIRRORS=(
    "mirrors.aliyun.com"
    "mirrors.tuna.tsinghua.edu.cn"
    "mirrors.ustc.edu.cn"
    "repo.huaweicloud.com"
    "mirrors.cloud.tencent.com"
)

for url in "${MIRRORS[@]}"; do
    # 使用 curl 测试连接到站点索引页的时间
    time=$(curl -o /dev/null -s -w '%{time_total}' --connect-timeout 2 "http://$url/ubuntu/")
    echo "$url - 响应时间: $time 秒"
done | sort -nk3
```
## 闭坑经验

### 安装无线网卡驱动

新显卡，特别是Realtek RTL8852BE，在社区和官方都没适配的情况下，还是推荐用替代办法。
#### 方案一：更换内置网卡（最推荐）

如果你的笔记本不是固化网卡（板载），建议直接更换 **Intel AX210** 或 **AX200**。

- **理由**: Intel 对 Linux 社区的支持是第一梯队的，驱动直接内置在内核中，无需任何手动安装，插上即用。
    

#### 方案二：免驱外接网卡

如果购买 USB 外接网卡，请务必搜索标注 **“免驱”** 或 **“支持树莓派/Linux”** 的型号。

- **推荐芯片**: MT7601U 或 RT5370（虽然速度慢，但极其稳定）。
    
- **避坑芯片**: 尽量避开 TP-Link 等低端型号中常见的 RTL8188 系列，它们在 Linux 下同样可能需要折腾编译。
