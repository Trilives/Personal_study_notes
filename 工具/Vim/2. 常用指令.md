#  Vim 高频常用指令速查笔记

## 一、 模式切换（核心核心）

* `i` : **Insert** 模式（在光标前开始输入）
* `Esc` : 退出当前模式，回到 **Normal** (命令) 模式
* `v` : **Visual** 模式（字符选择，类似鼠标拖动）
* `V` : **Visual Line** 模式（整行选择）
* `:` : **Command** 模式（执行存盘、退出、跳转等）

---

## 二、 移动光标（不用鼠标的秘诀）

### 1. 基础移动

* `h` `j` `k` `l` : 左、下、上、右
* `w` : 跳到下一个单词开头
* `e` : 跳到当前/下一个单词末尾
* `b` : 跳回上一个单词开头

### 2. 行内快速跳转

* `0` : 跳转到行首
* `$` : 跳转到行尾
* `f` + `字符` : 快速跳转到行内下一个该字符（例如 `f(` 直接跳到左括号）

### 3. 文件级跳转

* `gg` : 跳转到文件第一行
* `G` : 跳转到文件最后一行
* `:[n]` : 跳转到第 n 行（如 `:45` 跳到第 45 行，调试 Python 报错时必用）

---

## 三、 编辑与删除（算法重构神器）

* `x` : 删除当前光标下的字符
* `dd` : **剪切**当前行
* `[n]dd` : 剪切从当前行开始的 n 行
* `dw` : 删除一个单词
* `u` : **Undo**（撤销上一步操作）
* `Ctrl + r` : **Redo**（重做）
* `.` (点) : **重复**上一次编辑操作（极度好用）

---

## 四、 复制与粘贴（Yank & Put）

* `yy` : 复制当前行
* `y[motion]` : 结合移动指令复制（如 `y$` 复制到行尾，`yw` 复制单词）
* `p` : 粘贴到光标**后**
* `P` : 粘贴到光标**前**
* `yi(` : 复制括号内的所有内容（**Y**ank **i**nside **(**）
* `ci(` : 修改括号内的所有内容（删除并自动进入插入模式，改参数神技）

---

## 五、 搜索与替换

* `/关键词` : 向下搜索（按 `n` 跳到下一个，`N` 跳到上一个）
* `?关键词` : 向上搜索
* `:%s/旧词/新词/g` : 全文替换（`g` 表示一行内多次出现也替换）

---

## 六、 存盘与退出

* `:w` : 保存（Write）
* `:wq` : 保存并退出
* `:q!` : 强制退出，不保存修改
* `ZZ` : (命令模式下) 快速保存并退出

---

## 七、 针对 Python/ROS 开发的特殊指令

### 1. 代码缩进

* 在 Visual 模式下选中多行：
* `>` : 向右缩进（增加 Tab）
* `<` : 向左缩进（减少 Tab）



### 2. 分屏开发（查看算法时非常有用）

* `:sp [文件名]` : 水平分屏
* `:vsp [文件名]` : 垂直分屏（适合宽屏显示器，左边写代码，右边看 README 或参数文件）
* `Ctrl + w + w` : 在分屏窗口间切换

### 3. 粘贴模式（解决 Python 缩进混乱）

* `:set paste` : 进入粘贴模式（此时从外部粘贴不会乱序）
* `:set nopaste` : 退出粘贴模式
